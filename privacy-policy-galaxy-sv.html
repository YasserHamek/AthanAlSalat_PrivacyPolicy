Nätverksförfrågningar

Följande är den fullständiga listan över nätverksförfrågningar som appen kan göra. Inga direkta identifierare som ditt namn, e-postadress eller enhets-ID bifogas dessa förfrågningar:

  • Aladhan API (aladhan.com) — Används endast som reserv om beräkning av offlinebön misslyckas. Skickar dina GPS-koordinater och beräkningsmetod för att beräkna bönetider.
  • AlQuran Cloud API (alquran.cloud) — Hämtar koranöversättningar vid första användning. Skickar endast kapitelnummer och översättningskod.
  • Photon / Nominatim / OpenStreetMap — Används en gång för att omvandla dina GPS-koordinater till ett stadsnamn för visning. Ingen persistent identifierare skickas.
  • DNS-kontroll (dns.google) — En standard-DNS-sökning som används i bakgrunden för att kontrollera internettillgänglighet innan bönecachen uppdateras. Inga applikationsdata skickas.
