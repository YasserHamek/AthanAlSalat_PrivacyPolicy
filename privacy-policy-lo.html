ນະໂຍບາຍຄວາມເປັນສ່ວນຕົວຂອງ أذان الصلاة: ອາຊານ ອັລ ຊາລາດ

ອັບເດດລ່າສຸດ: ເດືອນເມສາ 2026

ສະຫຼຸບ: أذان الصلاة: ອາຊານ ອັລ ຊາລາດ ເຄົາລົບຄວາມເປັນສ່ວນຕົວຂອງເຈົ້າ. ພວກເຮົາບໍ່ໄດ້ເກັບກໍາຂໍ້ມູນສ່ວນບຸກຄົນໃນລະດັບບັນຊີເຊັ່ນ: ຊື່, ທີ່ຢູ່ອີເມວ, ຫຼືຕົວລະບຸການໂຄສະນາ. ບາງຄຸນສົມບັດທີ່ອີງໃສ່ສະຖານທີ່ອາດຈະສົ່ງພິກັດ GPS ຂອງທ່ານໄປຫາບໍລິການພາກສ່ວນທີສາມເພື່ອຄິດໄລ່ເວລາອະທິຖານຫຼືແກ້ໄຂຊື່ເມືອງ.

ຂໍ້ມູນທີ່ພວກເຮົາບໍ່ເກັບກຳ

أذان الصلاة: ອາຊານ ອັລ ຊາລາດ ບໍ່ໄດ້ເກັບກໍາຂໍ້ມູນສ່ວນບຸກຄົນໃນລະດັບບັນຊີ. ໂດຍສະເພາະ, ພວກເຮົາບໍ່ໄດ້ເກັບກໍາ:

  • ຊື່, ທີ່ຢູ່ອີເມວ, ຫຼືຂໍ້ມູນຕິດຕໍ່
  • ຕົວລະບຸອຸປະກອນເພື່ອການຕິດຕາມ
  • ການວິເຄາະການໃຊ້ງານ ຫຼືຂໍ້ມູນດ້ານພຶດຕິກຳ
  • ຂໍ້ມູນໃດໆເພື່ອຈຸດປະສົງໂຄສະນາ

ການເກັບຂໍ້ມູນໃນເຄື່ອງ

أذان الصلاة: ອາຊານ ອັລ ຊາລາດ ເກັບຂໍ້ມູນຕໍ່ໄປນີ້ຢູ່ໃນເຄື່ອງໃນອຸປະກອນຂອງທ່ານໂດຍໃຊ້ iOS UserDefaults. ການ​ຕັ້ງ​ຄ່າ​ເຫຼົ່າ​ນີ້​ໄດ້​ຖືກ​ບັນ​ທຶກ​ໄວ້​ໃນ​ອຸ​ປະ​ກອນ​, ເຖິງ​ແມ່ນ​ວ່າ​ຄຸນ​ສົມ​ບັດ​ທີ່​ອີງ​ໃສ່​ສະ​ຖານ​ທີ່​ຍັງ​ອາດ​ຈະ​ສົ່ງ​ພິ​ກັດ GPS ຂອງ​ທ່ານ​ກັບ​ການ​ບໍ​ລິ​ການ​ພາກ​ສ່ວນ​ທີ​ສາມ​ໃນ​ເວ​ລາ​ທີ່​ຈໍາ​ເປັນ​:

  • ທີ່ຕັ້ງ: ຈຸດພິກັດ GPS ແລະ ຊື່ເມືອງ, ຖືກຂຶ້ນຢູ່ໃນ cache ສຳລັບການຄຳນວນ ແລະ ສະແດງຜົນເວລາອ່ານນາທີ
  • ການຕັ້ງຄ່ານາທີ: ວິທີຄຳນວນ, ສຽງອະທານ, ການແຈ້ງເຕືອນ ແລະ ການຕັ້ງຄ່າອິຄອາມາ
  • Cache ນາທີ: Cache ເວລານາທີແບບອອບໄລນ໌ 30 ວັນ
  • ອັລກຸ່ອານ: ການຕັ້ງຄ່າການແປ, ການແປທີ່ຖືກ cache, ບຸກມາກ ແລະ ຂະໜາດຕົວໜັງສື
  • ການຕັ້ງຄ່າແອັບ: ພາສາ, ໂໝດມືດ ແລະ ສະຖານະການໜ້າ onboarding
  • ສະຖານະການຊື້: ບັນທຶກການຊື້ພາຍໃນແອັບ

ການຮ້ອງຂໍເຄືອຂ່າຍ

ຕໍ່ໄປນີ້ແມ່ນບັນຊີລາຍຊື່ຄົບຖ້ວນຂອງການຮ້ອງຂໍເຄືອຂ່າຍທີ່ app ອາດຈະເຮັດ. ບໍ່ມີຕົວລະບຸໂດຍກົງເຊັ່ນຊື່, ທີ່ຢູ່ອີເມວ, ຫຼື ID ອຸປະກອນຂອງທ່ານທີ່ຕິດກັບການຮ້ອງຂໍເຫຼົ່ານີ້:

  • Aladhan API (aladhan.com) — ໃຊ້ເປັນການຕອບແທນພຽງແຕ່ຖ້າການຄິດໄລ່ການອະທິຖານແບບອອບໄລນ໌ລົ້ມເຫລວ. ສົ່ງພິກັດ GPS ຂອງທ່ານແລະວິທີການຄິດໄລ່ເພື່ອຄິດໄລ່ເວລາອະທິຖານ.
  • AlQuran Cloud API (alquran.cloud) — ດຶງຂໍ້ມູນການແປອັລກຸ່ອານໃນການໃຊ້ງານຄັ້ງທຳອິດ. ສົ່ງສະເພາະໝາຍເລກບົດ ແລະ ລະຫັດການແປ.
  • Photon / Nominatim / OpenStreetMap — ໃຊ້ຄັ້ງດຽວເພື່ອປ່ຽນຈຸດພິກັດ GPS ຂອງທ່ານໄປເປັນຊື່ເມືອງສຳລັບສະແດງ. ບໍ່ມີຕົວລະບຸຖາວອນຖືກສົ່ງ.
  • ການກວດ DNS (dns.google) — ການຄົ້ນຫາ DNS ມາດຕະຖານທີ່ໃຊ້ໃນພື້ນຫຼັງເພື່ອກວດສອບການເຊື່ອມຕໍ່ອິນເຕີເນັດກ່ອນການໂຫຼດ cache ນາທີໃໝ່. ບໍ່ມີຂໍ້ມູນແອັບຖືກສົ່ງ.

ການຊື້ພາຍໃນແອັບ

أذان الصلاة: ອາຊານ ອັລ ຊາລາດ ສະເໜີການຊື້ພາຍໃນແອັບທາງເລືອກທີ່ດຳເນີນການທັງໝົດຜ່ານ Apple's App Store ໂດຍໃຊ້ StoreKit. ພວກເຮົາບໍ່ໄດ້ຮັບ ຫຼືເກັບຂໍ້ມູນການຊຳລະເງິນ. ບັນທຶກການຊື້ຖືກຈັດການໂດຍ Apple ແລະ ເກັບໄວ້ໃນເຄື່ອງຂອງທ່ານ.

ບໍລິການຂອງບຸກຄົນທີສາມ

أذان الصلاة: ອາຊານ ອັລ ຊາລາດ ບໍ່ໄດ້ລວມເອົາການວິເຄາະ, ໂຄສະນາ, ຫຼືບໍລິການລາຍງານການຂັດຂ້ອງຂອງບຸກຄົນທີສາມໃດໆ. ບໍ່ມີ SDK ສໍາລັບການວິເຄາະ ຫຼື ການໂຄສະນາ. ບໍລິການພາຍນອກດຽວທີ່ໃຊ້ສຳລັບການຊື້ແມ່ນ StoreKit ຂອງ Apple.

ຄວາມເປັນສ່ວນຕົວຂອງເດັກນ້ອຍ

أذان الصلاة: ອາຊານ ອັລ ຊາລາດ ບໍ່ໄດ້ເກັບກຳຂໍ້ມູນສ່ວນຕົວໃນລະດັບບັນຊີຈາກເດັກນ້ອຍ ຫຼືຜູ້ໃຫຍ່ໂດຍບໍ່ຮູ້ຕົວ.

ການລຶບຂໍ້ມູນ

ເນື່ອງຈາກຂໍ້ມູນທັງໝົດຖືກເກັບໄວ້ໃນເຄື່ອງຂອງທ່ານ:

  • ການຖອນການຕິດຕັ້ງແອັບຈະລຶບຂໍ້ມູນທີ່ເກັບໄວ້ໃນເຄື່ອງທັງໝົດ
  • ທ່ານສາມາດຣີເຊັດທີ່ຕັ້ງ ແລະ ການຕັ້ງຄ່າໄດ້ຈາກໜ້າຈໍການຕັ້ງຄ່າຂອງແອັບ

ການປ່ຽນແປງນະໂຍບາຍນີ້

ພວກເຮົາອາດອັບເດດນະໂຍບາຍຄວາມເປັນສ່ວນຕົວນີ້ເປັນໄລຍະ. ການປ່ຽນແປງໃດໆຈະຖືກໂພສໃນໜ້ານີ້ພ້ອມວັນທີ່ການດັດແກ້ທີ່ອັບເດດ.
