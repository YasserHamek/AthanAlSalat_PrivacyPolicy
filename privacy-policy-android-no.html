Nettverksforespørsler

Følgende er den komplette listen over nettverksforespørsler appen kan sende. Ingen direkte identifikatorer som navn, e-postadresse eller enhets-ID er knyttet til disse forespørslene:

  • Aladhan API (aladhan.com) — Brukes bare som en reserve hvis beregningen av frakoblet bønn mislykkes. Sender GPS-koordinater og beregningsmetode for å beregne bønnetider.
  • AlQuran Cloud API (alquran.cloud) — Henter Koranoversetninger ved første gangs bruk. Sender kun kapittelnummeret og oversettingskoden.
  • Photon / Nominatim / OpenStreetMap — Brukes én gang for å konvertere GPS-koordinatene dine til et bynavn for visning. Ingen permanent identifikator sendes.
  • DNS-sjekk (dns.google) — Et standard DNS-oppslag som brukes i bakgrunnen for å kontrollere internettilgjengelighet før bønnebufferen oppdateres. Ingen applikasjonsdata sendes.
