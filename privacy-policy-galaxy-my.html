နောက်ဆုံးအပ်ဒိတ်: ဧပြီ ၂၀၂၆

အကျဉ်းချုပ်: أذان الصلاة: အဇန် အယ်လ် ဆလာတ် သင်၏ privacy ကိုလေးစားပါ။ အမည်များ၊ အီးမေးလ်လိပ်စာများ သို့မဟုတ် ကြော်ငြာသတ်မှတ်မှုစနစ်များကဲ့သို့သော အကောင့်အဆင့် ကိုယ်ရေးကိုယ်တာဒေတာများကို ကျွန်ုပ်တို့ မစုဆောင်းပါ။ အချို့သော တည်နေရာအခြေခံအင်္ဂါရပ်များသည် ဆုတောင်းချိန်များကို တွက်ချက်ရန် သို့မဟုတ် မြို့အမည်များကို ဖြေရှင်းရန် သင်၏ GPS သြဒီနိတ်များကို ပြင်ပဝန်ဆောင်မှုများသို့ ပေးပို့နိုင်ပါသည်။

ကျွန်ုပ်တို့ မစုဆောင်းသော အချက်အလက်များ

أذان الصلاة: အဇန် အယ်လ် ဆလာတ် အကောင့်အဆင့် ကိုယ်ရေးကိုယ်တာ အချက်အလက်များကို မစုဆောင်းပါ။ အတိအကျအားဖြင့်၊ ကျွန်ုပ်တို့ မစုဆောင်းပါ-

  • နာမည်၊ အီးမေးလ် လိပ်စာ သို့မဟုတ် ဆက်သွယ်ရေး အချက်အလက်များ
  • ခြေရာခံခြင်း ရည်ရွယ်ချက်အတွက် စက်ပစ္စည်း သတ်မှတ်ပုံများ
  • အသုံးပြုမှု ပိုင်းခြားစိတ်ဖြာချက် သို့မဟုတ် အပြုအမူ ဒေတာများ
  • ကြော်ငြာ ရည်ရွယ်ချက်အတွက် မည်သည့် ဒေတာမျှ

ဒေသန္တရ ဒေတာ သိမ်းဆည်းခြင်း

أذان الصلاة: အဇန် အယ်လ် ဆလာတ် Android SharedPreferences ကို အသုံးပြု၍ သင့်စက်တွင် အောက်ပါဒေတာများကို စက်တွင် သိမ်းဆည်းပါ။ ဤဆက်တင်များကို စက်ပစ္စည်းပေါ်တွင် သိမ်းဆည်းထားသော်လည်း လိုအပ်သည့်အခါတွင် တည်နေရာအခြေခံလုပ်ဆောင်ချက်များက သင်၏ GPS သြဒီနိတ်များကို ပြင်ပဝန်ဆောင်မှုများသို့ ပေးပို့နိုင်သည်-

  • တည်နေရာ: ဆုတောင်းချိန် တွက်ချက်ခြင်းနှင့် ပြသခြင်းအတွက် cache ထားသော GPS좌표နှင့် မြို့နာမည်
  • ဆုတောင်း ဆက်တင်များ: တွက်ချက်မှု နည်းလမ်း၊ အဇာန် အသံများ၊ အကြောင်းကြားချက်နှင့် အိကာမာ နှစ်သက်ရာများ
  • ဆုတောင်း Cache: ၃၀ ရက် အော့ဖ်လိုင်း ဆုတောင်းချိန် cache
  • ကုရ်အာန်: ဘာသာပြန် နှစ်သက်ရာများ၊ cache ထားသော ဘာသာပြန်ချက်များ၊ မှတ်သားချက်များနှင့် စာလုံးအရွယ်အစား
  • အက်ပ် ဆက်တင်များ: ဘာသာစကား၊ မှောင်မိုက်မုဒ်နှင့် onboarding အခြေအနေ
  • ဝယ်ယူမှု အခြေအနေ: အက်ပ်အတွင်း ဝယ်ယူမှု မှတ်တမ်း

ကွန်ရက် တောင်းဆိုချက်များ

အောက်ပါတို့သည် အက်ပ်မှပြုလုပ်နိုင်သော ကွန်ရက်တောင်းဆိုမှုစာရင်း အပြည့်အစုံဖြစ်သည်။ ဤတောင်းဆိုမှုများတွင် သင့်အမည်၊ အီးမေးလ်လိပ်စာ သို့မဟုတ် စက် ID ကဲ့သို့သော တိုက်ရိုက် identifier များကို ပူးတွဲမထားပါ။

  • Aladhan API (aladhan.com) — အော့ဖ်လိုင်းဆုတောင်းချက် တွက်ချက်မှု မအောင်မြင်ပါကသာ လှည့်စားအဖြစ် အသုံးပြုသည်။ ဆုတောင်းချိန်များကို တွက်ချက်ရန် သင်၏ GPS သြဒိနိတ်များနှင့် တွက်ချက်နည်းကို ပေးပို့သည်။
  • AlQuran Cloud API (alquran.cloud) — ပထမဆုံး အသုံးပြုချိန်တွင် ကုရ်အာန် ဘာသာပြန်ချက်များကို ရယူသည်။ အခန်းနံပါတ်နှင့် ဘာသာပြန် ကုဒ်သာ ပို့သည်။
  • Photon / Nominatim / OpenStreetMap — ပြသခြင်းအတွက် သင့် GPS좌표ကို မြို့နာမည်အဖြစ် ပြောင်းလဲရန် တစ်ကြိမ် အသုံးပြုသည်။ ဆက်ထိန်းသော သတ်မှတ်ပုံ မပို့ပါ။
  • DNS စစ်ဆေးခြင်း (dns.google) — ဆုတောင်း cache ကို ပြန်လည်စုဆောင်းမည့်အပြင် အင်တာနက် ရရှိနိုင်မှုကို စစ်ဆေးရန် နောက်ခံတွင် အသုံးပြုသော စံနှုန်း DNS lookup ဖြစ်သည်။ အပလီကေးရှင်း ဒေတာ မပို့ပါ။

တတိယပါတီ ဝန်ဆောင်မှုများ

أذان الصلاة: အဇန် အယ်လ် ဆလာတ် သည် တတိယပါတီ ပိုင်းခြားစိတ်ဖြာချက်၊ ကြော်ငြာ သို့မဟုတ် ပျက်စီးမှု အစီရင်ခံ ဝန်ဆောင်မှုများကို မပေါင်းစပ်ပါ။ ခွဲခြမ်းစိတ်ဖြာမှု သို့မဟုတ် ကြော်ငြာ SDK မရှိပါ။ ဝယ်ယူမှုအတွက် အသုံးပြုသော တစ်ခုတည်းသော ပြင်ပ ဝန်ဆောင်မှုမှာ Samsung In-App Purchase (Samsung IAP) ဖြစ်သည်။
