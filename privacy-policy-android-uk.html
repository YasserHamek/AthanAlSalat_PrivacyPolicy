Коротко: أذان الصلاة: азан аль-салят поважає вашу конфіденційність. Ми не збираємо особисті дані на рівні облікового запису, такі як імена, адреси електронної пошти або рекламні ідентифікатори. Деякі функції на основі місцезнаходження можуть надсилати ваші GPS-координати стороннім службам для розрахунку часу молитви або визначення назв міст.

Інформація, яку ми не збираємо

أذان الصلاة: азан аль-салят не збирає особисту інформацію на рівні облікового запису. Зокрема, ми не збираємо:

  • Імена, адреси електронної пошти або контактну інформацію
  • Ідентифікатори пристроїв для цілей відстеження
  • Аналітику використання або поведінкові дані
  • Будь-які дані для рекламних цілей

Локальне зберігання даних

أذان الصلاة: азан аль-салят зберігає такі дані локально на вашому пристрої за допомогою Android SharedPreferences. Ці налаштування зберігаються на пристрої, хоча за потреби функції на основі визначення місцезнаходження також можуть надсилати ваші GPS-координати до сторонніх служб:

  • Місцезнаходження: GPS-координати та назва міста, збережені в кеші для розрахунку та відображення часу намазу
  • Налаштування намазу: Метод розрахунку, звуки азану, налаштування сповіщень та ікаме
  • Кеш намазу: 30-денний офлайн-кеш часу намазу
  • Коран: Налаштування перекладу, збережені в кеші переклади, закладки та розмір шрифту
  • Налаштування програми: Мова, темний режим та статус знайомства
  • Статус покупок: Запис покупок усередині програми

Мережеві запити

Нижче наведено повний список мережевих запитів, які програма може робити. До цих запитів не додаються прямі ідентифікатори, як-от ваше ім’я, електронна адреса чи ідентифікатор пристрою:

  • Aladhan API (aladhan.com) — Використовується як запасний варіант, лише якщо обчислення офлайн-молитви не вдається. Надсилає ваші GPS-координати та метод розрахунку для розрахунку часу молитви.
  • AlQuran Cloud API (alquran.cloud) — Отримує переклади Корану під час першого використання. Надсилає лише номер глави та код перекладу.
  • Photon / Nominatim / OpenStreetMap — Використовується один раз для перетворення ваших GPS-координат на назву міста для відображення. Постійний ідентифікатор не надсилається.
  • Перевірка DNS (dns.google) — Стандартний DNS-запит, що використовується у фоновому режимі для перевірки доступності Інтернету перед оновленням кешу намазу. Жодні дані програми не надсилаються.
