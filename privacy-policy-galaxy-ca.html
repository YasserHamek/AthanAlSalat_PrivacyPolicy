Emmagatzematge local de dades

أذان الصلاة: أذان الصلاة: أذان الصلاة: أذان الصلاة: أذان الصلاة: Athan al Salat emmagatzema les dades següents localment al dispositiu mitjançant Android SharedPreferences. Sol·licituds de xarxa

A continuació es mostra la llista completa de sol·licituds de xarxa que l'aplicació pot fer. No s'adjunta cap identificador directe com el vostre nom, adreça electrònica o identificador del dispositiu a aquestes sol·licituds:

  • Aladhan API (aladhan.com) — S'utilitza com a alternativa només si el càlcul de l'oració fora de línia falla. Envia les teves coordenades GPS i el mètode de càlcul per calcular els temps de pregària.
  • AlQuran Cloud API (alquran.cloud) — Obté traduccions de l'Alcorà en el primer ús. Envia només el número de capítol i el codi de traducció.
  • Photon / Nominatim / OpenStreetMap — S'utilitza una vegada per convertir les vostres coordenades GPS en un nom de ciutat per mostrar-lo. No s'envia cap identificador persistent.
  • Comprovació DNS (dns.google) — Una consulta DNS estàndard que s'utilitza en segon pla per comprovar la disponibilitat d'internet abans d'actualitzar la memòria cau de l'oració. No s'envien dades de l'aplicació.
