นโยบายความเป็นส่วนตัวของ أذان الصلاة: อาซาน อัล ซะละห์

อัปเดตล่าสุด: เมษายน 2026

สรุป: أذان الصلاة: อาซาน อัล ซะละห์ เคารพความเป็นส่วนตัวของคุณ เราไม่รวบรวมข้อมูลส่วนบุคคลระดับบัญชี เช่น ชื่อ ที่อยู่อีเมล หรือตัวระบุการโฆษณา คุณสมบัติตามตำแหน่งบางอย่างอาจส่งพิกัด GPS ของคุณไปยังบริการของบุคคลที่สามเพื่อคำนวณเวลาละหมาดหรือแก้ไขชื่อเมือง

ข้อมูลที่เราไม่เก็บรวบรวม

أذان الصلاة: อาซาน อัล ซะละห์ จะไม่รวบรวมข้อมูลส่วนบุคคลระดับบัญชี โดยเฉพาะอย่างยิ่ง เราไม่รวบรวม:

  • ชื่อ ที่อยู่อีเมล หรือข้อมูลติดต่อ
  • ตัวระบุอุปกรณ์เพื่อวัตถุประสงค์ในการติดตาม
  • การวิเคราะห์การใช้งานหรือข้อมูลพฤติกรรม
  • ข้อมูลใดๆ เพื่อวัตถุประสงค์ในการโฆษณา

การจัดเก็บข้อมูลในเครื่อง

أذان الصلاة: อาซาน อัล ซะละห์ เก็บข้อมูลต่อไปนี้ไว้ในอุปกรณ์ของคุณโดยใช้ iOS UserDefaults การตั้งค่าเหล่านี้จะถูกบันทึกไว้ในอุปกรณ์ แม้ว่าคุณสมบัติตามตำแหน่งอาจส่งพิกัด GPS ของคุณไปยังบริการของบุคคลที่สามเมื่อจำเป็น:

  • ตำแหน่ง: พิกัด GPS และชื่อเมืองที่แคชไว้เพื่อคำนวณและแสดงเวลาละหมาด
  • การตั้งค่าละหมาด: วิธีการคำนวณ เสียงอะซาน การแจ้งเตือนและค่าการตั้งค่าอิกอมะห์
  • แคชละหมาด: แคชเวลาละหมาดแบบออฟไลน์ 30 วัน
  • อัลกุรอาน: ค่าการตั้งค่าการแปล การแปลที่แคชไว้ บุ๊กมาร์ก และขนาดตัวอักษร
  • การตั้งค่าแอป: ภาษา โหมดมืด และสถานะการเริ่มต้นใช้งาน
  • สถานะการซื้อ: บันทึกการซื้อในแอป

คำขอเครือข่าย

ต่อไปนี้เป็นรายการคำขอเครือข่ายทั้งหมดที่แอปอาจสร้างขึ้น ไม่มีการแนบตัวระบุโดยตรง เช่น ชื่อ ที่อยู่อีเมล หรือรหัสอุปกรณ์ของคุณมากับคำขอเหล่านี้:

  • Aladhan API (aladhan.com) — ใช้เป็นทางเลือกเฉพาะในกรณีที่การคำนวณคำอธิษฐานแบบออฟไลน์ล้มเหลว ส่งพิกัด GPS ของคุณและวิธีการคำนวณเพื่อคำนวณเวลาละหมาด
  • AlQuran Cloud API (alquran.cloud) — ดึงการแปลอัลกุรอานในการใช้งานครั้งแรก ส่งเฉพาะหมายเลขบทและรหัสการแปล
  • Photon / Nominatim / OpenStreetMap — ใช้ครั้งเดียวเพื่อแปลงพิกัด GPS ของคุณเป็นชื่อเมืองสำหรับแสดงผล ไม่มีการส่งตัวระบุถาวร
  • การตรวจสอบ DNS (dns.google) — การค้นหา DNS มาตรฐานที่ใช้ในพื้นหลังเพื่อตรวจสอบความพร้อมใช้งานของอินเทอร์เน็ตก่อนรีเฟรชแคชละหมาด ไม่มีการส่งข้อมูลแอปพลิเคชัน

การซื้อในแอป

أذان الصلاة: อาซาน อัล ซะละห์ เสนอการซื้อในแอปที่ไม่บังคับซึ่งดำเนินการทั้งหมดผ่าน App Store ของ Apple โดยใช้ StoreKit เราไม่ได้รับหรือจัดเก็บข้อมูลการชำระเงินใดๆ บันทึกการซื้อได้รับการจัดการโดย Apple และจัดเก็บในเครื่องบนอุปกรณ์ของคุณ

บริการของบุคคลที่สาม

أذان الصلاة: อาซาน อัล ซะละห์ ไม่ได้รวมบริการวิเคราะห์ โฆษณา หรือรายงานข้อผิดพลาดของบุคคลที่สามใดๆ ไม่มี SDK การวิเคราะห์หรือโฆษณาใดๆ บริการภายนอกเดียวที่ใช้สำหรับการซื้อคือ StoreKit ของ Apple

ความเป็นส่วนตัวของเด็ก

أذان الصلاة: อาซาน อัล ซะละห์ จะไม่รวบรวมข้อมูลส่วนบุคคลระดับบัญชีจากเด็กหรือผู้ใหญ่โดยเจตนา

การลบข้อมูล

เนื่องจากข้อมูลทั้งหมดจัดเก็บในเครื่องบนอุปกรณ์ของคุณ:

  • การถอนการติดตั้งแอปจะลบข้อมูลที่จัดเก็บในเครื่องทั้งหมด
  • คุณสามารถรีเซ็ตตำแหน่งและค่ากำหนดได้จากหน้าจอการตั้งค่าของแอป
