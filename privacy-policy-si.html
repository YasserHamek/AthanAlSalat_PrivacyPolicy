ජාල ඉල්ලීම්

යෙදුම සිදු කළ හැකි ජාල ඉල්ලීම්වල සම්පූර්ණ ලැයිස්තුව පහත දැක්වේ. ඔබගේ නම, ඊමේල් ලිපිනය, හෝ උපාංග හැඳුනුම්පත වැනි සෘජු හඳුනාගැනීම් මෙම ඉල්ලීම්වලට අමුණා නැත:

  • Aladhan API (aladhan.com) — නොබැඳි යාච්ඤා ගණනය කිරීම අසාර්ථක වුවහොත් පමණක් පසුබැසීමක් ලෙස භාවිතා වේ. යාච්ඤාවේ වේලාවන් ගණනය කිරීමට ඔබගේ GPS ඛණ්ඩාංක සහ ගණනය කිරීමේ ක්‍රමය යවයි.
  • AlQuran Cloud API (alquran.cloud) — පළමු භාවිතයේදී කුරාන් පරිවර්තන ලබා ගනී. අධ්‍යාය අංකය සහ පරිවර්තන කේතය පමණක් යවයි.
  • Photon / Nominatim / OpenStreetMap — ඔබේ GPS좌표 ප්‍රදර්ශනය සඳහා නගරයේ නමකට පරිවර්තනය කිරීමට එක් වරක් භාවිතා කෙරේ. ස්ථිර හඳුනාගැනීමක් නොයවයි.
  • DNS පරීක්ෂාව (dns.google) — නමාස් ගබඩාව නැවුම් කිරීමට පෙර අන්තර්ජාල ලබාගැනීම පරීක්ෂා කිරීමට පසුබිමේ භාවිතා කෙරෙන සාමාන්‍ය DNS සෙවීමකි. යෙදුම් දත්ත නොයවයි.
