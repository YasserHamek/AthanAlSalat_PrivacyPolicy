Mrežni zahtjevi

Slijedi potpuni popis mrežnih zahtjeva koje aplikacija može postaviti. Nikakvi izravni identifikatori kao što su vaše ime, adresa e-pošte ili ID uređaja nisu priloženi ovim zahtjevima:

  • Aladhan API (aladhan.com) — Koristi se kao zamjena samo ako izračun izvanmrežne molitve ne uspije. Šalje vaše GPS koordinate i metodu izračuna za izračunavanje vremena molitve.
  • AlQuran Cloud API (alquran.cloud) — Preuzima prijevode Kur'ana pri prvom korištenju. Šalje samo broj poglavlja i kod prijevoda.
  • Photon / Nominatim / OpenStreetMap — Koristi se jednom za pretvaranje vaših GPS koordinata u naziv grada za prikaz. Ne šalje se nikakav trajni identifikator.
  • DNS provjera (dns.google) — Standardni DNS upit koji se koristi u pozadini za provjeru dostupnosti interneta prije osvježavanja predmemorije molitve. Ne šalju se nikakvi podaci aplikacije.
