Verkkopyynnöt

Seuraavassa on täydellinen luettelo verkkopyynnöistä, joita sovellus voi tehdä. Näihin pyyntöihin ei liitetä suoria tunnisteita, kuten nimeäsi, sähköpostiosoitettasi tai laitetunnusta:

  • Aladhan API (aladhan.com) — Käytetään varavaihtoehtona vain, jos offline-rukouslaskenta epäonnistuu. Lähettää GPS-koordinaatit ja laskentamenetelmäsi rukousaikojen laskemiseksi.
  • AlQuran Cloud API (alquran.cloud) — Hakee Koraanin käännökset ensimmäisellä käyttökerralla. Lähettää vain suuran numeron ja käännöskoodin.
  • Photon / Nominatim / OpenStreetMap — Käytetään kerran GPS-koordinaattien muuntamiseen kaupungin nimeksi näyttämistä varten. Pysyvää tunnistetta ei lähetetä.
  • DNS-tarkistus (dns.google) — Tavallinen DNS-haku, jota käytetään taustalla internetin saatavuuden tarkistamiseen ennen rukoiluvälimuistin päivittämistä. Sovellusdataa ei lähetetä.
